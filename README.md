# Maritime-Intelligence-System-
Satellite-to-Economics Analystics Platform

Turning Satellite Data into Economic Intelligence

I recently built a Maritime Intelligence System that uses SAR satellite data (Sentinel-1) to analyze global shipping activity and extract economic signals.

🔍 What it does:
• Detects ships using radar imagery (independent of AIS)
• Identifies major shipping lanes using clustering
• Maps global shipping density and congestion
• Analyzes lagged relationships between shipping activity and oil prices
• Builds a Maritime Risk Index combining traffic + financial signals

📊 Why this matters:
Maritime traffic is one of the strongest indicators of global trade and energy movement. By using satellite data, we can monitor this activity even when traditional tracking systems fail or are manipulated.
